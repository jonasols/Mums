# Mums

### Requirements:
- NodeJS
- NPM
- Truffle (npm i -g truffle)


### Running project from console:
- truffle compile
- truffle develop
- migrate


### Running tests from console:
- truffle test ./test/TestMumsToken.sol
- truffle test ./test/mumstoken.js
